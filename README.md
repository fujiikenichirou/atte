# アプリケーション名
「 atte 」
![atte](./localhost_.png)

## アプリケーション概要
勤怠管理システムです。  
出勤は1日1回で日を跨ぐとリセットされます。  
休憩は何回でも取れます。


## アプリケーションURL
https://github.com/fujiikenichirou/atte.git


##　機能一覧
- ログイン機能
- 勤怠打刻
- 日付別勤怠情報取得


## 使用技術
- php 7.4.9
- laravel 8.x
- docker-compose 3.8
- mysql
